⚡ Como atualizar os dados

Os dados são atualizados automaticamente todo dia às 06h BRT via GitHub Actions.

Para forçar uma atualização manual agora:

  1. Acesse o repositório no GitHub
  2. Clique em Actions → 🔄 Sync Meta Ads
  3. Clique em Run workflow
  4. Opcionalmente defina o período (ex: 30 dias)
  5. Aguarde ~1 minuto e recarregue a página

Ou clique em ⚡ Sincronizar agora na barra de ferramentas para ir direto ao GitHub Actions.
